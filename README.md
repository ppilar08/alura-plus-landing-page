🚀 Alura Plus - Landing Page
Uma landing page moderna inspirada no Alura Plus, desenvolvida durante o treinamento da Alura. O projeto apresenta uma interface bem estruturada, com foco em organização de layout, estilização e boas práticas de desenvolvimento front-end.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica dos elementos da página.
CSS3: Estilização utilizando variáveis, grid layout e efeitos de hover para uma interface mais moderna.
Google Fonts: Utilização da fonte Inter para melhor legibilidade e estética.

🌟 Funcionalidades Principais
Interface Responsiva: Layout organizado e adaptável para diferentes tamanhos de tela.
Seções Informativas: Estrutura dividida em blocos para melhor experiência do usuário.
Estilização Moderna: Uso de cores, espaçamentos e tipografia alinhados com boas práticas de design.
Efeitos Visuais: Interações com hover em botões e elementos da página.

🎓 Créditos e Aprendizado
Este projeto foi desenvolvido como parte de um estudo guiado em parceria com a Alura. Durante o desenvolvimento, foram praticados conceitos fundamentais de HTML e CSS, como organização de código, uso de variáveis CSS, grid layout e construção de interfaces modernas e bem estruturadas.
